Задача по предсказанию совершения пользователем 
целевого действия на сервисе СберАвтоподписка (классификация)

Общее описание задачи:

Построить модель, предсказывающую совершение целевого действия
(ориентировочное значение ROC-AUC ~ 0.65) — факт совершения
пользователем целевого действия.
Датасет представлен двумя таблицами характеристики пользователей и 
совершаемые ими действия.
Модель должна быть интерпретируемой.


Общие этапы работы над проектом:

1.) Обработка исходных таблиц: удаление пустых и неинформативных значений, 
выделение факта целевого действия, перевод остальных значений в числовые 
или категориальные (с уменьшением числа категорий).
2.) Склейка таблиц по 'session_id' и финальная их обработка: удаление аномалий и выбросов,
приведение значение целевой переменной к формату 0/1.
Разбивка датафрейма на X, y.
3.) Обработка X: числовые столбцы - MinMaxScaler(), категориальные - OHEncoding.
4.) Моделирование - логистическая регрессия (т.к. требуется интерпретируемость)
с настройками по умолчанию, обучение модели и сохранение в файл.
5.) Загрузка модели, предсказание и перевод вероятности в классы с порогом 0,03 
(на тестовой выборке такой порог наилучшее качество показал).
6.) Сохранение результатов предсказания в файл.
